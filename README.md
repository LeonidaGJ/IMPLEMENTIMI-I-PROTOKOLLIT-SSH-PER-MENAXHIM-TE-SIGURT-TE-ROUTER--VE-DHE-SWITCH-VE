# IMPLEMENTIMI-I-PROTOKOLLIT-SSH-PER-MENAXHIM-TE-SIGURT-TE-ROUTER--VE-DHE-SWITCH-VE
SSH (Secure Shell) është nje protokoll I cili ofron komunikim të inkreptuar dhe autentifikim të sigurt ndërmjet klientit dhe paisjes.Përmes SSH, administratorët mund të aksesojnë dhe menagjojnë paisjet e rrjetit në një mënyrë të sigurt, duke reduktuar rrezikun e sulmeve kibernetike.
